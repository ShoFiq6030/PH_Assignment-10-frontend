# HomeNest - Real Estate Platform

Welcome to **HomeNest**, a modern real estate platform where users can browse, search, and filter properties to find their dream homes. This project showcases frontend development skills using **React.js**, **Tailwind CSS**, and **Axios** for API communication.

---

## Features

* **Property Listings:** View a curated collection of properties with images, descriptions, and key details like price, rooms, and area.
* **Search Functionality:** Search properties by name with a responsive debounced search bar.
* **Filter & Sort:** Filter properties by category, price, and posting date. Sort listings by price or newest/oldest.
* **Responsive Design:** Fully responsive layout that works seamlessly on mobile, tablet, and desktop devices.
* **User-Friendly UI:** Clean and intuitive interface with dark mode support for better user experience.

---

## Tech Stack

* **Frontend:** React.js, Tailwind CSS,daisyUi
* **API Requests:** Axios
* **Hosting:** Vercel
* **State Management:** React Hooks (useState, useEffect, useContext, useRef)

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/ShoFiq6030/PH_Assignment-10-frontend.git
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Run the development server:

   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) to view in the browser.
